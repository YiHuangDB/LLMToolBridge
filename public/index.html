🚀 Quick Start Guide

ℹ️
Welcome to LLM Tool Bridge!
This proxy enables tool/function calling for any LLM by converting OpenAI-style tool calls into prompts.

Getting Started:

  1. Configure a Target: Go to "LLM Targets" and add your LLM endpoint
  2. Test Connection: Use the Playground to test your setup
  3. Secure Your API: Set up API keys in the Security section
  4. Integrate: Use the API Documentation to integrate with your client

🔐 API Key Management

ℹ️
API Security:
• Admin Interface: Always open access - no authentication required
• Chat API Endpoint: Optional API key for /v1/chat/completions endpoint
• Set a Chat API key if you want to restrict access to the proxy service

Chat API Security

No API key required (open access)
If set, clients must provide this key in the Authorization header when calling /v1/chat/completions

⚙️ Proxy Settings

Comma-separated list of allowed origins, or * for all

API Endpoints

▶ POST /v1/chat/completions Create chat completion with tool calling
Request Body Schema
Field Type Required Description
model string Required Model ID to use
messages array[Message] Required Array of messages
tools array[Tool] Available tools/functions
stream boolean Stream response (SSE)
temperature number Sampling temperature (0-2)
max_tokens integer Maximum tokens to generate
Example Request
{
  "model": "gpt-3.5-turbo",
  "messages": [
    {"role": "system", "content": "You are a helpful assistant."},
    {"role": "user", "content": "What's the weather in San Francisco?"}
  ],
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "get_weather",
        "description": "Get current weather",
        "parameters": {
          "type": "object",
          "properties": {
            "location": {"type": "string"}
          },
          "required": ["location"]
        }
      }
    }
  ],
  "stream": false
}
