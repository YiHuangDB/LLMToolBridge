Configure Your Client

Point your OpenAI-compatible client to:

API Endpoint: http://localhost:3500/v1/chat/completions
Model: Configure a target first
API Key: Any value (required by some clients)

Example Clients:

  • ChatBox
  • Continue.dev
  • Open WebUI
  • Any OpenAI SDK

Python Example

from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:3500/v1",
    api_key="any-key"
)

response = client.chat.completions.create(
    model="your-model",
    messages=[
        {"role": "user", "content": "Hello!"}
    ],
    tools=[...]  # Tools will be converted automatically
)

cURL Example

curl http://localhost:3500/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "your-model",
    "messages": [{"role": "user", "content": "Hello!"}],
    "stream": true
  }'
